JSX Transpilation - JSX가 JavaScript로 변환되는 과정 Virtual DOM - 선언적 UI를 가능하게 하는 핵심 개념 Reconciliation - O(n³)에서 O(n)으로 최적화한 비교 알고리즘 얕은 비교와 불변성 - React가 선택한 트레이드오프 Fiber Architecture - 더 나은 사용자 경험을 위한 새로운 엔진

얕은 비교 vs 깊은 비교

얕은 비교 (React)

// 참조만 비교
Object.is(prevState, nextState)

비교 방식

깊은 비교 (React가 안 함)

// 전체 객체 내용 비교
JSON.stringify(prevState) === JSON.stringify(nextState)

React는 === 대신 Object.is를 사용합니다

• NaN === NaN: false
• Object.is(NaN, NaN): true

얕은 비교 장점: 빠른 성능 (O(1)), 예측 가능, 명시적 코드
